## ✨ Features

- **Real-time Weather**: Current temperature, condition, feels-like, humidity, wind speed, and more
- **Beautiful UI**: Built entirely with Jetpack Compose for a modern, responsive design
- **Location Support**: Search by city name or use device location
- **Detailed Forecast**: Hourly and daily weather forecasts
- **Clean Architecture**: Well-structured MVVM pattern with repository layer
- **Offline Support**: Basic caching of last fetched weather data
- **Material You Design**: Dynamic theming and smooth animations

## 🛠️ Tech Stack

- **Language**: Kotlin
- **UI**: Jetpack Compose
- **Architecture**: MVVM + Clean Architecture
- **Networking**: Retrofit + OkHttp + Moshi (or Kotlin Serialization)
- **Dependency Injection**: Hilt
- **Async**: Kotlin Coroutines + Flow
